## 🏆 Featured Projects

### 🤖 1. Molecubes — Modular Robotic Arm
<div align="center">
  <img src="https://raw.githubusercontent.com/EmbeddedDreamTeam/molecubes/refs/heads/main/assets/logo.png" alt="Molecubes Logo" width="600">
</div>

A modular robotic arm consisting of interconnected, 3D-printed cubes. Each module contains a servomotor and magnetic connectors for hot-swappable assembly and rapid reconfiguration.

* **Architecture:** Every cube is powered by an **ESP32-C3** microcontroller.
* **Communication:** Every module is connected to the near ones through full-duplex UART and the user interact with the arm with a python script.
* **My Role:** Hardware design (CAD/3D printing) and embedded firmware development.

📦 **[Explore the Molecubes Repository](https://github.com/EmbeddedDreamTeam/molecubes)**
